Red ROVOR, Red ROVOR, send Data right over

The Red ROVOR project is a project to automate and stream the astronomical reduction process as much as possible for the Remote Observatory for Variable Object Research (ROVOR). It is currently composed of two sub-projects, the backend which consists of a python code-base that performs the reduction operations, and a web interface which allows the user to send commands to the backend. In the future we expect to build on this and provide interaction with the ROVOR telescope control as well.
